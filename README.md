# PokeApp
Demo app using MVVM architecture and the standard Android Kotlin stack!

## Libraries Used:
- OkHttp: API calls
- Retrofit: API to data classes
- Coroutines: multithreading
- Koin: dependency injection
- Coil: image loading
- RecyclerView: displaying lists
